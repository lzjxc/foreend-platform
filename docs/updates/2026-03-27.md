# 2026-03-27 开发追踪模块 (Dev Tracker)

## 新增内容

### 开发追踪模块 `/dev-tracker`

单页 8 tab 布局的开发追踪看板，用于跟踪项目里程碑、任务、活动记录和 Claude Code 会话。

**8 个 Tab:**
1. **Dashboard** - 总览统计卡片 (活跃里程碑、任务状态分布、本周活动、未关联活动) + 活动趋势图 + 服务分布 + 开发时间线
2. **Milestones** - 里程碑列表 (状态筛选、应用筛选、展开详情、完成操作、进度条)
3. **Apps** - 应用管理 (分类筛选、展开详情、分类标签管理)
4. **Specs** - 需求规格 (阶段筛选、里程碑筛选、展开详情/关联任务、阶段前进/回退/取消/同步)
5. **Tasks** - 任务管理 (状态/类型/服务/里程碑筛选、展开详情、完成/回退操作、优先级标签)
6. **Activities** - 活动记录 (服务/类型/来源筛选、展开详情、批量选择+关联到任务)
7. **Sessions** - Claude Code 会话 (服务/状态筛选、展开摘要、查看对话内容、重新分析)
8. **Config Backups** - 配置备份 (文件变更 diff、manifest 查看)

### 交互特性

- 所有列表项支持展开/折叠查看详情
- Session 支持查看完整对话内容 (conversation viewer)
- Task 支持完成/回退状态操作
- Spec 支持阶段管理 (advance/retreat/cancel/sync)
- Activities 支持批量选择 + 关联到任务 (bulk-link)
- Milestone 支持完成操作 + 进度条显示
